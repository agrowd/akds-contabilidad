# AKDs Command Center - Map

## Architecture
- **Framework**: Next.js 16 (App Router)
- **Database**: SQLite (`local.db`)
- **UI System**: Vanilla CSS + Glassmorphism
- **Logic**: Server Actions (`lib/actions.ts`)

## Key Modules
- `/alumnos`: Student management & Matrix
- `/cobros`: Financial registry & WhatsApp audit
- `/rendicion`: Reconciliation with Excel imports
